# Robôs Tupiniquins

![](https://i.imgur.com/4rshAzr.gif)

## Introdução

Robôs foram enviados para marte, e a AEB pediu para fazer um programa que faça eles andarem a partir de comandos, é isso que verá aqui!

## Funcionalidades

- **Estabelecer o tamanho da malha**: A malha/grid é escolhida pelo próprio usuário;
- **Posicionamento inicial**: O posicionamento inicial do robô é escolhido pelo usuário;
- **Movimentação detalhada**: A movimentação é estabelecida com direção cardeal;
- **Validações**: Caso você erre em algo, tem validação para ver o erro.

## Como Utilizar

1. Clone o repositório ou baixe o código fonte.
```
git clone https://github.com/Gustav0W/RoboTupiniquim-2025
```
2. Abra o terminal ou o prompt de comando e navegue até a pasta raíz.
```
cd RoboTupiniquim-2025
```
3. Utilize o comando abaixo para restaurar as dependências do projeto.
```
dotnet restore
```
4. Em seguida compile com o comando: 
```
dotnet build --configuration Release
```

5. Execute com o comando:
```
dotnet run
```

6. Você verá esse terminal:
```
===========================
     Robôs Tupiniquins
===========================
Informe os valores que representarão a malha (X Y): 
```
